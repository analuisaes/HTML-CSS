Pseudo-Elementos

Pseudo-Elemento, pode agir nas classes, no elemento, no id, eles mexem no conteúdo periférico do elemento. Pseudo-Elementos vai ser sempre (::)

Alguns símbolos que podem ser usados nas CSS para criar seletores personalizados

  • # = id
  • . = class
  • : = pseudo-class
  • :: = pseudo-elementos
  • > = children (filho)
