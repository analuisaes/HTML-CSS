Inserindo vídeos hospedados localmente

No html5, os padroes mais utilizados e mais suportados, são os formatos: MP4 M4V, WEBM, OGV. O ideal é que se tenha vários arquivos de vários formatos.

Dica de Ouro: Se o vídeo for grande, converter ele para menor no aplicativo HandBrake

Este vídeo está hospedado no meu próprio servidor.

Qual tag colocar? >video< e os parâmetros width para tamanho (recomendado 500), controls para aparecer o controle

Para o site com toda certeza abrir em vários navegadores diferentes, é importamte colocar vários formatos de vídeo em ordem crescente, dentro da tag <video controls>, colocar a tag >source src type<

Inserindo vídeos do YouTube

Para inserir vídeos do YouTube, bata ir no vídeo que vocee deseja, selecionar "compartilhar" e depois "incorporar", Já vai ser te dado o código em html pronto para ser colocado no seu site. Aí não consome o site do seu servidor, mas sim, do YouTube, economizando bastante pro seu cliente e para você. Uma coisa legal é criar um canal no YouTube e colocar o vídeo desejado lá. Lembre-se que se deve colocar vídeos com direitos autorais. Um site legal para hospedar vídeos também, além do YouTube, é o Vimeo, nele é possível escolher se você quer que o vídeo seja público ou privado, diferente do YouYube, que só é possivel pegar um código de vídeo se o mesmo estiver público, lembrando que o vimeo é pago mas tem bem mais profissionalização. No YouTube aparece também outros vídeos como opção assim que o vídeo é pausado, já no Vimeo não. No vímeo tem limitação de videos para poderem ser usados. NO YouTube qualquer pesssoa pode ver seu vídeo. Lembre-se sempre dos Direitos Autorais.
