Código fonte / Pré formatado

Código fonte são comandos. Por exemplo: o comando document.getelements.id(teste) é um comando escrito, em linguagem Java Script. Essa fonte não reserva o mesmo espaço lateral para cada letra. Usando a tag <code><code> é possível deixar o comando numa forma mais simples de ser lido. Veja: document.getelements.id(teste). A fonte monoespaçada serve para escrever códigos com todas as letras do mesmo tamanho facilitando a leitura.

Quando se usa a tag <pre>, é considerado todos os espaços do jeitinho que eu coloco e organizo neste programa. por exemplo:

Sem a <tag> pre e só coma tag <code>:

#include <iostream> using namespace std; int main () { int value, pow, result=1; cout << "Por favor insira o operando:" << endl; cin >> value; cout << "Por favor insira o expoente:" << endl; cin >> pow; for (int cnt=0; cnt!=pow; cnt++) result*=value; cout << value << " à potência de " << pow << " é: " << result << endl; return 0; }

Citações Simples (Sem quebra)

Para colocar aspas (""), basta utilizar a tag <q><q>, dá para colocar as aspas manualmente, mas a tag dá um sentido para a citação. Quando for uma citação, é importante que se use a tag <q><q>, mas quando não for, pode ser colocado as aspas manualmnete mesmo.

Quem não se enfeita, por si, se rejeita.

Citações complestas (com quebras)

Para uma citação mais completa, é utilizada a tag <blockquote><blockquote>, essa tag dá quebra de linha e deslocamento lateral. Pode ser adicionado na tag, um link doo endereço da citação, usando cite, visualmente, não acontece nada, mas os sistemas, sejam eles qual for, vai saber que essa frase ta referenciando o dono da citação. Por exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Officiis cumque nesciunt nobis dolorem nam sequi corporis natus, eaque sapiente saepe officia explicabo illo laborum recusandae quasi! Ex reiciendis commodi consequuntur!
