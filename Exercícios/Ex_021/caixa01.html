Exemplo de caixa box-level

Parágrafos também são exemplos de box-level, mas os links são exemplos de caixas inline-level, vamos ver como tudo isso funciona.

Usando DevTools no google

No google, clicar com o botão direito, clicar em inspecionar. Vai abrir o DevTools, o mesmo, geralmente abre em elementos, para o que vai ser feito, ir em Elementos, aparece o modelo de ciaxas do site. Quando você clica em algum elemento, na parte de baixo, da para ver todas as configuraç~oes de CSS que já exixstem para ele, mesmo os que são feitos autamticamente. User agente, é o navegador que está exibindo o conteúdo, todo navegador tem uma configuração básica de CSS que é seguida por todos eles

Explicando algumas função do estilo padrão de CSS deste site aqui(no caso do h1).

        
            h1 {
                display: block; /*um eleemnto do tipo box-level, o display pode ser mudado por outros valores*/
                font-size: 2em; /*Duas vezes o tamanho original*/
                margin-block-start: 0.67em;
                margin-block-end: 0.67em;
                margin-inline-start: 0px;
                margin-inline-end: 0px;
                font-weight: bold;
            }

É possível alterar as configurações do site direto pelo DevTools, sem ter que ir para o arquivo original, deve-se selecionar o elento, clicar na propriedade que é para ser alterada, lá em baixo em styles, ái da para alterar na mão, colocando o valor desejado, ou girar a rodinha do mouse, aí altera do jeito que a gente quer. Isso não altera o arquivo original, só o arquivo visível.

Lá no DevTools, na aba "computed", é possível ver o display, se é block para box-level ou se é inline para inline-level. Isso é alterável, se quiser alterar, basta dentro do seletor no CSS colocar display:inline; ou display:block;. Quando é inline não dá para fazer altura e largura. É possível também fazer display:inline-block;, assim ele vai ser inline mas vai ter características de bloco, podendo ser mudado o tamanho, altura, largura, entre outras coisas.
