Estilos Locais / Internos

Para a área do cabeçalho, que é a tag <head>, colocar a tag <style> pode ser embaixo de <title> mesmo. Este método é configurado por seletores, então dentro da tag <style>, vamos por qual tag queremos estilizar, por exemplo, essa configuração que fiz neste site:

        
            <style>
                body{
                    background-color: rgb(255, 163, 179);
                    font-family: Arial, Helvetica, sans-serif;
                    font-size: 20px;
                }
                h1{
                    color: rgb(108, 108, 216);
                    background-color: rgb(167, 167, 247);
                }
                h2{
                    color: rgb(226, 88, 88);
                }
                p{
                    text-align: justify;
                }
            </style>

Dsvantagens do CSS Local/Interno: Geralmente, o estilo ocupa muito mais linhas que o coneúdo. Se tiver 30 páginas e eu quiser usar o mesmo estilo, vou ter que copiar o estilo e colar nas 30 páginas.

Esta técnica é útil para usar quando vai criar um estilo só.

    Algumas tags em CSS

  • "color:cor;" --> Para colorir a fonte
  • "background-color:cor;" (dentro do body) --> Para colorir o fundo da página
  • "font-family:fonte;" --> Para definir a fonte
  • "text-align:comando;" --> Para arrumar o texto de forma centralizada, justificada, mais para esquerda ou mais para a direita
  • "font-size: px;" --> Para aumentar o tmanho da fonte base
  • "font-size: em;" --> Para diminuir o tamanho da fonte
  • "background-color:cor;" (dentro dos títulos e subtítulos ou textos) --> Para mudar a cor de fundo do texto.
