Principais formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag <b></b> (não semântica - não tem significado).

Nessa frase temos um termo em destaque usando a tag <strong></strong> (semântica, tem significado de destaque).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica).

Nessa frase temos um termo em ênfase usando a tag <em> (semântica).

IMPORTANTE:

Semântica é algo com significado, recomendado o uso no HTML. Não semântica é algo só com forma recomendado o uso no CSS.

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

Para mudar a cor do marcador, como é uma coisa específica, pode simplesmente ser usado dentro da tag <mark> um parâmetro style e selecionando a cor. (usar essa técnica mais para coisa específica)

Para colocar uma maior parte de palavras selecionadas em destaque, o ideal é dar um comando só, para isso, tem que usar o CSS com o comando <style>, dentro da tag <head.>

Texto grande e pequeno

Para criar um texto grande usamos as tags <big><big>. Já para criar um texto pequeno, usamos a tag <small><small>

Texto deletado

POdemos marcar um texto como excluido, para indicar que que ele ppode ser lido, mas não considerado usando as tags <del><del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase de que ele fo adicionado depois, usando as tags <ins><ins>

tag também é sublinhada mas não é como a tag <u><u>. A tag <del><del>é semântica (com significado) e a tag <u><u> é não semântica (Não tem significado).

Texto subescrito

Para iserir coisas do tipo X2*3 para elevar o x ao quadrado, devemos usar a tag <sup><sup>

Texto sobreescrito

Já para coisas do tipo H2O, para colocar o 2 sobre o H, devemos utilizar a tag <sub><sub>
