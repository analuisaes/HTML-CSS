Imagem Dinâmica

É muito importante adaptar o tamanho da imagem no tamanho da tela do usuário. A tag <picture> permite que você coloque imagens. Para uma imagem não ficar muito maior do que deveria na tela do pc, dentro ta tag <picture>, coloque uma tag <source:media:sizes>.Dentro dos parênteses,coloque max, e depois dos dois pontos, se a tela do pc for menor que 1000px, ele vai carregar a foto m. Assim: <source media=(max-width: 1000px)"...>

Tente abrir esse site em vários dispositivos diferentes ou aumente e diminua o tamanho do seu navegador.

A tag <source:media:sizes> tem a seguinte função: o site vai carregar a tela g(grande), mas, se o tamanho da tela for menor que 1000px(caso necessário aumente este número mais 50.), dentro de source é colocada a imagem m(média). e também pode ser colocada a imagem p(pequena). As imagens devem estar em ordem crescente.

Reproduzindo Áudio

Para colocar áudio em HTML5, deve-se usar a tag <audio> e os parâmetros controls(para o controle de áudio aparecer) e autoplay(para reproduzir o áudio)

Existem muitos navegadores que não são compátiveis com o MP3, então pode ser utilizado outros formatos de arquivo, como MPEG, OGG e WAV. Mas, para isso ser possível, deve ser usado o formato MP3. Se você quiser ter compatibilidade 100% com todos os navegadores, é importante ter outras formas de adicionar o áudio para dar para o site todas as chances para ser possível que ele seja reproduzido. A ordem de tentativas deve ser crescente. Abaixo, tem uma opção de baixar o arquivo caso o navegador do usuário não consiga carregar o arquivo que deveria, então, é para ser colocada essa opção de baoxar dentro da tag <audio>. Para ativar o autoplay, é importante por o parâmetro preload dentro da tag <audio>.
Esta, tem 3 opções:

  • Auto --> Ele só vai considerar que o site terminou de carregar, depois que carregar todo o áudio também.(não indicado);
  • Metadata --> Vai carregar só o nome do arquivos e outras coisas;
  • None --> Não vai carregar nada, só quando o usuário apertar play no controle.
Pode ser colocado auto se tiver o parâmetro autoplay.
