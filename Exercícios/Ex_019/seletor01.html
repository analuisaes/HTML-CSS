Seletor id

Para colocar um seletor ID em determinado elemento, quando tem muitos elementos iguais, basta, dentro da tag, colocar uma identificação. Ex:

<h1 id="principal">Criando sites com HTML e CSS</h1>

Aí, dentro do CSS externo, fica assim:

        
            /*h1 sem identificação*/
            h1{
                color: #7cc083;
            }
            /*h1 que foi identificado com o nome principal*/
            #principal{
                color: #13678A;
                text-align: center;
            }

Seletor class

Quando eu tenho 2 elementos e quero aplicar a mesma configuração nos dois, não posso usar id, vou usar class. Que também faz configuração personalizada, mas eu posso colocar vários elementos com a mesma classe, ao contrário do id que posso colocar só um. IMPORTANTE: Não dê nome para suas classes pela forma mas sim pela funcionalidade.É possível também colocar mais de uma class no mesmo elemento. basta separar com espaço. como está no h2 css avançado. Também pode colocar class junto com id, mas o id sobrepõe as configurações de class. Como colocar class, ex:

>h2 class="intermediario"<HTML Intermediário>/h2<

Dentro do css, fica assim:

        
            .intermediario{
                color: #cddd41;
            }
