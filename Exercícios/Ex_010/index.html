Usando links externos

É um link que vai daqui para o site de outra pessoa.

Para um link externo, você envelopa a frase desejada para "virar um link" e digita a, vai se tornar um código <a href="link">Nome do link</a> ex:

Você pode acessar o jogo Pegaxy

Para o link ser aberto em outra janela e não apagar o seu site, basta colocar os parâmetros targe=_"blank" -- Para abrir em outra janela e rel="external" -- Para mostrar que é um link externo.(Isso é importante para quando o outro site não é seu)

Usando links internos

É um link que vai daqui para outro site seu.

Deve-se criar outro arquivo dentro da mesma pasta do arquivo padrão. ex:

Está é a primeira página do site, se você quiser pode acessar também, a minha segunda página.

Já nesse tipo de link, é importante usar parâmetro rel="next" para indicar que vai para a segunda página

Você também pode acessar a nossa página de notícias

Quando os links são internos, não é necessário que coloque o parâmetro target="_blank", porque não tem nessecidade da página ser aberta em outro site, aí, ou você não coloca nada, ou coloca target="_self"
