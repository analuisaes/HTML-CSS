Listas Ordenadas

Para criar uma lista ordenada, devemos usar a tag <ol> para ordenar e dentro da tag <ol>, devemos colocar uma tag <li> que são os itens da lista. Não existe orbigação para colocar a tag </li>. Com a type, é possível ter opções de formatação da lista, (1 A a I i). E com start, posso colocar o que eu quero no topo da lista.

  1. Acordar
  2. Tomar banho
  3. Comer
  4. Sair
  5. Ir para o curso
  6. Ir para casa
  7. Estudar
  8. Dormir

Lista de definição

Para criar uma lsta de definição, vamos usar a tag <dt> e a tag <dd>. A tag dt é o termo, e a tag dd é a definição do termo. Ex:

HTML
Linguagem de marcação para a criação de conteúdo de um site.
CSS
Linguagem de marcação para a criação de design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
